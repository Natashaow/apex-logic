## Design Philosophy

The interface functions strictly as a high-density command center instrument panel. Data density, scannability, and thin-line alignment replace superficial visual fluff.

**Three guiding principles:**
1. **Signal over noise** — Color appears only when it communicates systemic risk or state change.
2. **Density over decoration** — Every pixel earns its place by conveying operational data.
3. **Human at the apex** — The interface is an instrument that amplifies human judgment, never replaces it.
