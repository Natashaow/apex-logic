# AGENTS.md

This project's working rules for AI coding agents live in **`CLAUDE.md`** at the repository root. Read that file first, then follow its read-first sequence:

1. `APEX_LOGIC_PLAN.md`
2. `memory-bank/ACTIVE_CONTEXT.md` (including the **In Progress** section)
3. `memory-bank/PROGRESS.md`
4. `memory-bank/DECISIONS.md`
5. `memory-bank/COMPONENT_MAP.md` (before editing any component or token)

Cursor-native sessions are additionally governed by `.cursor/rules/apex-context.mdc` and `.cursor/rules/change-protocol.mdc`, which encode the same intent.

**Do not** propose or apply changes before reading `CLAUDE.md`.
